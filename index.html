What is netboot.xyz?

netboot.xyz is a collection of iPXE scripts that allow you to rapidly network boot Operating Systems, Utilities and other tools very easily. It allows you the flexibility of booting installations without having to go track down and download installation media. It's especially useful for remote access environments when you don't want to utilize remote attach CD in a Dell DRAC, HP iLO or some other type of remote tool.

Check out the new netboot.xyz documentation hosted at ReadTheDocs:

  • http://netbootxyz.readthedocs.org/en/latest/

Grab these bootloaders and drop them in Fusion/VirtualBox/DRAC/iLO to get started:

  • ISO: dhcp | static
  • Floppy/USB: dhcp | static
  • Linux Bootable Kernel: dhcp | static
